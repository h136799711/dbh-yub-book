### 1.2 创建充值订单

**版本历史**

|service_version版本号|service_type服务名称|说明|
|----|---|---|
|100|by_RechargeOrder_create|接口创建|

**业务参数**

|参数 |类型|是否必须|备注|
| ---------------- | ------------------------ | ------------------------ | ------------------------ |
|amount|integer|是|单位:分|
|method|string|是|支付方式,1.1的code参数值|

**返回参数** 
```
{
    "_cost": "588081us",
    "_start": 1604026557.000589,
    "code": 0,
    "data": 1, //充值订单唯一码
    "msg": "操作成功",
    "notify_id": "666"
}
```
