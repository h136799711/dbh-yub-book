### 1.2 地址添加


**版本历史**

|service_version版本号|service_type服务名称|说明|
|----|---|---|
|100|by_UserAddress_create|接口创建|

**业务参数**

|参数 |类型|是否必须|备注|
| ---------------- | ------------------------ | ------------------------ | ------------------------ |
|contactMobile|string|是|联系手机号18位以内|
|name|string|是|姓名|
|detail|string|是|120长度以内|
|is_default|integer|是|是否默认地址 1: 是 0: 否|

**返回参数** 
```
{
    "_cost": "716158us",
    "_start": 1603952344.641252,
    "code": 0,
    "data": "",
    "msg": "操作成功",
    "notify_id": "666"
}
```
