### 1.2 理财产品停售(停售表示用户无法再购买，已购买到期可赎回)

**版本历史**

|service_version版本号|service_type服务名称|说明|
|----|---|---|
|100|by_KjdsFund_close|接口创建|

**业务参数**

|参数 |类型|是否必须|备注|
| ---------------- | ------------------------ | ------------------------ | ------------------------ |
|id|integer|是|产品id|

**返回参数** 
```
{
    "data": "",
    "code": 0,
    "msg": "操作成功"
}
```
